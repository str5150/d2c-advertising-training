# D2C広告研修講座

## 概要
D2C事業における広告運用の知識とスキルを習得するための研修講座です。インハウス運用の重要性から、実践的なPDCAサイクルの構築まで、体系的に学習できます。

## 講座構成

### 事前テスト
- **広告運用知識テスト**: 50問の選択式テストで現在の知識レベルを測定
- 名前入力機能付き
- 結果の保存・管理機能
- 間違えた問題の詳細表示

### 講座内容
- **第0章**: はじめに：なぜ、今インハウス運用なのか？
- **第1章**: 誰に、どんな場所で、どんなタイミングで？
- **第2章**: ルールや仕組みを知る
- **第3章**: 記事とクリエイティブディレクション
- **第4章**: PDCAを回す方法

## 使用方法

### 1. 事前テストの実施
1. `test.html` にアクセス
2. お名前を入力
3. 50問のテストを実施
4. 結果を確認し、知識レベルを把握

### 2. 講座の受講
1. `index.html` にアクセス
2. 各章を順番に学習
3. ワークシートを活用して実践

## GitHub Pages での公開

### 1. リポジトリの作成
1. GitHubで新しいリポジトリを作成
2. このフォルダの内容をアップロード

### 2. GitHub Pages の設定
1. リポジトリの「Settings」タブを開く
2. 「Pages」セクションに移動
3. Source を「Deploy from a branch」に設定
4. Branch を「main」に設定
5. 「Save」をクリック

### 3. アクセス方法
- メインページ: `https://[ユーザー名].github.io/[リポジトリ名]/`
- 事前テスト: `https://[ユーザー名].github.io/[リポジトリ名]/test.html`

**現在の公開URL（例）:**
- 事前テスト: `https://str5150.github.io/d2c-advertising-training/test.html`

### 4. アクセス制限について
- 事前テストページは、GitHub Pagesの公開URLからのみアクセス可能です
- 検索エンジンにインデックスされないよう設定されています（noindex）
- 他のサイトから埋め込まれたり、意図しない場所からアクセスされることを防ぎます

## 機能説明

### 事前テスト機能
- **50問の選択式問題**: 広告運用の基礎から応用まで幅広くカバー
- **リアルタイム進捗表示**: 現在の問題数と進捗率を表示
- **タイマー機能**: テスト開始からの経過時間を表示
- **結果保存機能**: ローカルストレージに結果を保存
- **知識レベル判定**: 正解率に応じて4段階のレベルを表示
- **間違い問題の詳細表示**: 間違えた問題の解説を表示
- **結果一覧機能**: 過去のテスト結果を確認可能

### 講座機能
- **レスポンシブデザイン**: モバイル・タブレット・PC対応
- **進捗管理**: 各章の完了状況をローカルストレージで管理
- **インタラクティブ要素**: ツールチップ、展開可能セクション等
- **ワークシート**: 各章の演習用ワークシート

## 技術仕様
- **HTML5**: セマンティックなマークアップ
- **CSS3**: モダンなスタイリング、レスポンシブデザイン
- **JavaScript (ES6+)**: モジュール化されたコード
- **LocalStorage**: データの永続化
- **GitHub Pages**: 静的サイトホスティング

## カスタマイズ

### テスト問題の追加・編集
`test-data.js` ファイルの `testQuestions` 配列を編集してください。

### スタイルの変更
- メインページ: `assets/css/style.css`
- テストページ: `test-style.css`

### 講座内容の編集
各章のHTMLファイルを編集してください。
